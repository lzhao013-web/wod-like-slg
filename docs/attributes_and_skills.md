# WOD-like 人物属性与技能系统

本版本把原先“攻击 / 防御 / 速度 / 命中 / 闪避”的扁平人物面板，扩展为更接近网页游戏 WOD 思路的“八维属性 + 技能检定 + 职业技能树”结构。核心目标是让配队、站位、装备、等级与副本机制之间形成更多构筑关系。

## 八维属性

| 属性 | 定位 | 主要影响 |
| --- | --- | --- |
| 力量 strength | 近战压制 | 生命、攻击、近战技能收益 |
| 体质 constitution | 生存 | 生命、防御、防护技能收益 |
| 灵巧 dexterity | 技巧命中 | 命中、闪避、盗贼/战士技巧 |
| 敏捷 agility | 行动效率 | 速度、闪避、盗贼/游侠技能 |
| 智力 intelligence | 奥术 | 法术伤害、法力上限、战术/诅咒 |
| 意志 willpower | 抗压与信仰 | 法力上限、牧师/守卫技能 |
| 感知 perception | 射击与洞察 | 命中、游侠/战术技能 |
| 魅力 charisma | 神术与鼓舞 | 治疗、信仰、队伍支援 |

## 派生属性

后端在 `effective_stats` 中根据八维属性生成派生值：

- `hp_bonus`：力量 + 体质 × 2。
- `attack_bonus`：力量与灵巧提供少量基础攻击。
- `defense_bonus`：体质与力量提供基础防御。
- `speed_bonus` / `accuracy_bonus` / `evasion_bonus`：分别来自敏捷、感知、灵巧组合。
- `mana` / `max_mana`：法力资源。角色拥有当前法力与法力上限，法力上限主要由智力、意志、魅力推导。部分法术、治疗、净化与祝福技能会消耗法力；法力不足时战斗 AI 会跳过该技能，改用其他可用技能或普通攻击。
- 每日推进会像生命一样恢复一部分法力；休整日会完全恢复法力。
- `melee_score`、`ranged_score`、`finesse_score`、`magic_score`、`faith_score`、`healing_score`、`guard_score`、`tactics_score`：技能检定倾向。

## 技能焦点

每个技能可声明 `attribute_focus` 与 `attribute_scale`。战斗结算时，技能会根据角色八维计算额外伤害、治疗或命中收益。

技能还可以声明 `mana_cost`。非消耗型战技通常为 0；牧师与法师的核心技能会消耗法力，使“是否连续施法、是否休整、是否保留高阶法术”成为长期推进的一部分。

| focus | 典型职业 | 主要属性 |
| --- | --- | --- |
| melee | 战士 | 力量、灵巧、敏捷 |
| ranged | 游侠 | 感知、灵巧、敏捷 |
| finesse | 盗贼 | 灵巧、敏捷、感知 |
| magic | 法师 | 智力、意志、感知 |
| faith | 牧师 | 意志、魅力、智力 |
| healing | 牧师 | 魅力、意志、智力 |
| guard | 守卫 / 战士 | 体质、意志、力量 |
| tactics | 战术姿态 | 感知、智力、魅力 |

## 职业技能深度

六个职业现在都有 7-8 个技能，按等级逐步解锁：

- 战士：保护、重击、顺劈、盾击、战斗专注、斩杀。
- 守卫：保护、盾墙、盾击、挑衅怒吼、钢铁誓言、壁垒光环。
- 游侠：瞄准射击、箭雨、猎人标记、牵制射击、穿甲箭、暴雨箭幕。
- 盗贼：背刺、破甲、淬毒刃、烟步、肾击、影袭终结。
- 牧师：治疗术、净化、圣佑祷言、群体治疗、驱邪圣击、群体净化、神圣庇护。
- 法师：奥术箭、火球术、奥术屏障、引燃、连锁闪电、脆弱诅咒、陨火雨。

## 新状态与构筑点

新增或强化了以下状态：

- 正面：`focus`、`might`、`barrier`、`ward`、`evasion_up`、`regeneration`。
- 负面：`marked`、`vulnerable`、`enfeeble`、`slow`。
- 原有：毒、流血、燃烧、诅咒、破甲、眩晕、护卫、盾墙。

构筑示例：

- 游侠 `猎人标记` + 盗贼 `影袭终结`：标记后提高终结爆发。
- 盗贼 `破甲` + 战士 `重击 / 斩杀`：处理高防怪。
- 牧师 `圣佑祷言` + 守卫 `壁垒光环`：应对毒、诅咒、火焰 Boss。
- 法师 `脆弱诅咒` + 游侠 `穿甲箭`：对重甲精英做双重破防。
